Шаг 1: Подготовка файла
    touch secure_file.txt
    chmod 600 secure_file.txt
    # Теперь только владелец (вы) может читать/писать этот файл

Шаг 2: Запуск без setuid бита   
    ./main secure_file.txt

    Вывод:

    Реальный UID: 1000
    Эффективный UID: 1000

    Пытаемся открыть файл 'secure_file.txt'...
    Файл успешно открыт!

    === ПОСЛЕ УСТАНОВКИ ID ===
    Реальный UID: 1000
    Эффективный UID: 1000

    Пытаемся открыть файл 'secure_file.txt'...
    Файл успешно открыт!

Шаг 3: Установка setuid бита
    sudo chown root main    # Делаем root владельцем программы
    sudo chmod u+s main    # Устанавливаем setuid бит
    ls -l main             # Проверяем:должно быть -rwsr-xr-x

Шаг 4: Запуск с setuid битом
    ./main secure_file.txt

    Вывод:
    === ДО УСТАНОВКИ ID ===
    Реальный UID: 1000
    Эффективный UID: 0

    Пытаемся открыть файл 'secure_file.txt'...
